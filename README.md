** Run the code:**

**1.**
`Install node version 16.0.0`

**2.**
`cd build`

**3.**
`npm install -g pnpm`

**4.**
`pnpm install && pnpm build`

or (if above doesn't work)

`npm install --legacy-peer-deps && npm run build`

**5.**
`pnpm cypress open`
